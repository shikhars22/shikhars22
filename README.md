## 📈 Featured MLOps & Data Engineering Projects

### 🎥 1. YouTube Sentiment Insights (End-to-End MLOps Pipeline)
*An enterprise-grade, fully automated NLP pipeline that scrapes YouTube comments, trains a multi-class sentiment analysis model using XGBoost, and deploys it to production.*
- **Pipeline Orchestration**: Leveraged **DVC (Data Version Control)** to build reproducible pipelines across data acquisition, preprocessing, feature extraction, and training.
- **Experiment Tracking**: Integrated **MLflow** for robust tracking of hyperparameters, validation metrics (Accuracy, F1-Score), and training logs.
- **Model Registry & Lifecycle**: Configured MLflow Model Registry to systematically transition models from *Staging* to *Production* with integrated automated test verifications.
- **Containerization & Scale**: Packaged the pipeline and REST API using **Docker** and deployed it to **Kubernetes** to ensure high-availability and horizontal scalability.
- **Stack**: Python, XGBoost, DVC, MLflow, Docker, Kubernetes, AWS.

### 🔄 2. Reproducible ML Pipelines with DVC
*Designed and built a modular data engineering pipeline using DVC to solve reproducibility issues caused by data drift.*
- Formulated custom pipeline steps (`dvc.yaml`) for parallelized data cleaning, feature extraction, and evaluations.
- Set up automated remote storage caching using S3/GCS buckets to enable fast, secure, and incremental collaborative data engineering.
- **Stack**: DVC, Git, AWS S3, Google Cloud Storage, Python, Pandas.

### 🧪 3. MLflow Experimentation & Run Tracker
*An advanced model lifecycle management system ensuring high visibility of neural network and machine learning model training.*
- Developed custom MLflow logging hooks to capture system metrics (CPU/GPU utilization), hyperparameters, confusion matrices, and ROC-AUC charts.
- Configured dynamic Slack/Email webhooks to alert engineering teams whenever a newly registered model outperforms the production baseline.
- **Stack**: MLflow, Python, PyTorch, Scikit-learn, Webhooks.
